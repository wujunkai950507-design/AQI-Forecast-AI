# 空氣品質預測系統（AQI Forecast Demo）
一個結合智慧預測概念與環境工程應用的空氣品質指數（AQI）展示平台（Demo）

---

## 一、作品介紹（Overview）

本系統為空氣品質預測與風險提示平台（Demo）。  
使用者可輸入溫度、濕度、PM2.5 與 CO₂ 等環境數據，系統會以 **PM2.5 作為 AQI 推估的主要依據**，並搭配溫濕度作為情境修正參考，輸出 **AQI 估計值** 與對應的空氣品質等級（良好 / 普通 / 不良）。

> 說明：CO₂ 主要用於室內通風與舒適度判讀，**不屬於 AQI 官方污染物項目**，在本 Demo 中作為輔助參考資訊呈現。

本專案用於展示智慧環境監測、污染風險判讀與預警提示的應用情境。

---

## 二、作品功能（Features）

- 輸入環境數據（溫度、濕度、PM2.5、CO₂）
- 推估 AQI 估計值（以 PM2.5 為主）
- 自動判斷空氣品質等級（良好 / 普通 / 不良）
- 以顏色標示風險狀態，提升辨識度
- 可作為環境監測系統的前端 Demo（可擴充串接感測器或 Open Data / API）

---

## 三、系統流程（Flow）

```text
使用者輸入環境數據（溫度／濕度／PM2.5／CO₂）
            ↓
AQI 推估（以 PM2.5 為主，溫濕度作參考修正）
            ↓
輸出 AQI 估計值
            ↓
判斷空氣品質等級（良好／普通／不良）
            ↓
顯示結果於畫面（顏色提示）
