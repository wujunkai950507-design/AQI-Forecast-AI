# 空氣品質預測系統（AQI Forecast System）
一套結合環境指標分析與資料運算的空氣品質預測平台

---

## 📌 作品介紹（Overview）

本系統透過分析多項環境因子（溫度、濕度、PM2.5、CO₂），
計算對應的空氣品質指數（AQI），並自動判斷空氣品質等級，
以視覺化方式呈現結果，協助使用者快速理解當前環境狀態與風險程度。

本作品可作為環境監測、校園空氣品質展示或智慧環境系統之前端示範平台。

---

## ✨ 系統功能（Features）

- 輸入環境數據（溫度、濕度、PM2.5、CO₂）
- 即時計算 AQI 預測值
- 自動判斷空氣品質等級（良好 / 普通 / 不良）
- 以顏色標示環境風險狀態
- 提供清楚直覺的空氣品質顯示介面

---
